To visualize step by step:
type in terminal
`python visualize <level> <name_file_input> <algorithm_name> `

With level is '1', '2' or 'advance'.
name_file_input: '1', '2', '3'....
algorithm_name: bfs, dfs, ucs, gbfs1, gbfs2, astar1, astar2, adv...

EG: if you want run a visualization of astar algorithm with heuristic-function-1 on input1.txt of level_1, type in your terminal:
```python -m visualize 1 1 astar1```
